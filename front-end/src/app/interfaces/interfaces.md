# Directorio `interfaces`

## Propósito
Define los contratos de datos mediante interfaces y tipos TypeScript que garantizan la consistencia tipada en toda la aplicación.

## Convenciones de nombres
| Tipo | Sufijo | Ejemplo |
|------|--------|---------|
| Interface | `.interface.ts` | `user.interface.ts` |
| DTO | `.dto.ts` | `create-user.dto.ts` |
| Enum | `.enum.ts` | `user-role.enum.ts` |
| Type | `.type.ts` | `id.type.ts` |
